# otsliced module

Installation
  Silent install can be done with the switch /S (e.g. openturns-otsliced-0.0-py27.exe /S).
  To test otsliced module installation, in an OpenTURNS python console type:
    import otsliced
    print otsliced.__version__

Uninstallation
  otsliced module will be automatically uninstalled at OpenTURNS removal.
  otsliced module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
